# Talora Branding

## Dirección
Talora se define como una marca de `operational premium`: sobria, clara y seria, con color solo donde mejora la lectura operativa.

La base visual es:
- blanco
- gris
- negro

Los colores pastel se reservan para:
- métricas
- estados
- highlights contextuales
- bloques de lectura rápida

## Base neutra
- `Ink / negro principal`: `#1C1D22`
- `Texto fuerte`: `#111318`
- `Gris medio`: `#6B7280`
- `Gris suave`: `#9AA1AE`
- `Borde`: `#E2E4EC`
- `Superficie fría`: `#F5F6FA`
- `Fondo general`: `#F8F9FC`
- `Blanco`: `#FFFFFF`

## Pasteles de dato
- `Lilac`: `#EFE9FF`
- `Sky`: `#E7F5FB`
- `Sand`: `#F7EDDF`
- `Mint`: `#E8F6EB`
- `Rose`: `#F8EAEF`

## Reglas de uso
- `80-90%` de la interfaz debe vivir en neutros.
- Los pasteles no se usan como branding estructural.
- El shell, sidebar, topbar, tablas y contenedores base deben mantenerse en neutros.
- Los pasteles solo entran en KPI cards, badges, paneles de contexto y acentos de lectura.
- El CTA principal usa tinta/negro.
- El CTA secundario usa blanco con borde gris.
- No usar verde pantano ni verdes saturados como color principal de la plataforma.

## Tipografía
- `UI / body`: `IBM Plex Sans`
- `Display / títulos`: `Fraunces`

## Sensación de marca
- menos “startup decorada”
- más “herramienta operativa premium”
- calma visual
- datos claros
- color con intención
